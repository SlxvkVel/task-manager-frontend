# Менеджер задач 

Клиентская часть веб-приложения менеджера задач с интегрированной системой уведомлений и напоминаний. Разработано в рамках курсовой работы по дисциплине «Фронтенд-разработка».

Ссылка на github pages: https://slxvkvel.github.io/task-manager-frontend/

## Технологии

- HTML5, CSS3 (БЭМ)
- JavaScript (ES6+), модули, классы
- Vite (сборка)
- localStorage (хранение задач)
- Notification API (уведомления)
- Akamai Time API (точное серверное время)
- Jest (юнит-тестирование)

## Функциональность

- Добавление, редактирование, удаление задач
- Отметка о выполнении
- Фильтрация (все / активные / выполненные)
- Установка срока выполнения и напоминания
- Браузерные уведомления о наступлении времени напоминания
- Отдельная страница со списком всех напоминаний
- Страница статистики с круговой диаграммой выполнения задач
- Отображение точного серверного времени (Akamai Time API) с обновлением каждую секунду
- Выделение просроченных задач (красная рамка)
- Адаптивный дизайн (десктоп, планшет, телефон)

## Установка и запуск

```bash
npm install
npm run dev
```
